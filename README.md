# Healthcare VLAN & Inter-VLAN Routing Lab

## Overview
This project simulates a segmented healthcare network using VLANs and inter-VLAN routing.

## Features
- VLAN segmentation
- Trunk configuration
- Inter-VLAN communication
- Multilayer switch routing
- Connectivity testing using ping

## VLANs
- VLAN 10 → Radiology Department
- VLAN 20 → Doctor Department

## Devices Used
- Cisco 2960 Switch
- Cisco 3560 Multilayer Switch
- PCs representing CT, PACS, Doctor

## Validation
Successful ping tests between VLAN devices confirmed proper routing and connectivity.
